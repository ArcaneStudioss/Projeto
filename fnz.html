Conclusão

    Nós utilizamos diversos meios de pesquisa para atingir o resultado esperado, com isso, nosso projeto de instalação de placas solares na escola foi um sucesso na teoria, também confeccionamos um site para explicação do conteúdo, se o projeto das placas solares na escola for um sucesso fora do papel, futuramente todas as escolas do Brasil poderão aderir este tipo de geração própria de energia.

Técnicas de pesquisa

    Para comprovarmos nossa hipótese sobre os itens citados acima iremos utilizar os métodos de pesquisa:


  • Pesquisa Bibliográfica - Pesquisa em livros.
  • Pesquisa Experimental - Experimentos para comprovar análises.
  • Pesquisa Quantitativa - Coleta de dados numerais.
  • Pesquisa Qualitativa - Entrevistas, análise de conteúdo não numeral.
